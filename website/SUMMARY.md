## 📞 SUPPORT & TROUBLESHOOTING

### Quick Fixes

**"Site doesn't load in Odoo"**
→ Check: URL is correct in ODOO_INTEGRATION.html
→ Check: Site deployed successfully
→ Check: Using HTTPS (not HTTP)

**"Styles are broken"**
→ Check: `.nojekyll` file exists
→ Check: All files committed to git
→ Check: Browser console for errors

**"Forms don't work"**
→ Check: Iframe sandbox allows forms
→ Test: Form works outside iframe first
→ Check: CORS settings

**Need more help?**
→ See "Troubleshooting" section in `DEPLOYMENT.md`
