## 📋 NEXT STEP: Add to Odoo (5 minutes)

### 1. Create New Page in Odoo

1. Log into your **Odoo** instance
2. Go to **Website** app
3. Click **+ New** → **Page**
4. Enter:
   - **Page Title:** `Workforce 360°` (or your preferred name)
   - **URL:** `/workforce-360`
5. Click **Create**

---

### 2. Add Integration Code

1. On the new page, click **Edit**
2. Look for the **< />** icon (HTML editor) in the toolbar
   - Or click **Insert** → **Code** → **HTML**
3. **Delete all default content** in the editor
4. Open the file: **`ODOO_INTEGRATION.html`** from your project
5. **Copy EVERYTHING** (Ctrl+A, then Ctrl+C)
6. **Paste** into the Odoo HTML editor (Ctrl+V)
7. Click **Save**

---

### 3. Create Menu Item

1. Go to **Website** → **Configuration** → **Menus**
2. Click **Create**
3. Fill in:
   - **Menu:** `Workforce 360°`
   - **URL:** `/workforce-360` (match the page URL from step 1)
   - **Parent Menu:** `Top Menu`
   - **Sequence:** 10 (or adjust for position)
4. Click **Save**

---

### 4. Publish & Test

1. Go back to your page
2. Click **Publish** button (top right)
3. Open your Odoo website in a new tab
4. You should see **"Workforce 360°"** in the menu
5. **Click it!** 🎉

## 🔧 If You Have Issues

**Blank iframe or "Refused to connect":**
- Wait 30 seconds for initial load
- Check browser console (F12) for errors
- Ensure you're using HTTPS Odoo instance

**Styles look broken:**
- Clear browser cache (Ctrl+Shift+R)
- Check that all CSS/JS files loaded (F12 → Network tab)

**Forms don't submit:**
- This is expected - forms are demo/static
- You'll need to connect to a backend later
